# Servicio MCP

## Descripción General

Este es un servicio simple que implementa el estándar MCP (Model Context Protocol) de Anthropic. Proporciona una herramienta para consultar en tiempo real los valores del dólar estadounidense en diversas monedas utilizando una API pública.

## Datos por los que puede preguntar

- **Tasas de Cambio en Tiempo Real**: Consulta los valores actuales del USD en diferentes monedas.
- **Descubrimiento de Herramientas**: Pregunta sobre las herramientas disponibles para ver su información.
- **Información de Actualización**: Verifica la hora de la última actualización de los datos.
- **Resúmenes de Precios**: Obtén resúmenes de precios en varias monedas para verificar que la herramienta funcione correctamente.

## Preguntas posibles

Puedes interactuar con el servicio preguntando cosas como:

- "¿Qué herramientas tienes disponibles?" para ver la información de las herramientas.
- "¿Cuál es la hora de la última actualización?" para verificar la frescura de los datos.
- "Proporciona un resumen de precios en diferentes monedas" para confirmar que la herramienta está funcionando correctamente.
